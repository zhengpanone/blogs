=============================================
5.5 pdb 调试
=============================================

pdb 是基于命令行的调试工具，类似于gnu的gdb(调试C/C++)

执行时调试
==================

::

 python -m pdb some.py

命令 l,n,c

- **l** list 展示当前代码
- **n** next 向下执行一行代码
- **c** continue 继续执行代码
- **b** break 添加断点 b+行数,查看断点
- ***clear* clear1 删除断点
- **s** step 进入到一个函数
- **p** print 打印一个变量的值
- **a** arg 打印所有参数
- **q** quit 退出调试
